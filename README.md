 E-Commerce Cart Application

An e-commerce web application which has cart functionality 
---

 Features

* Product listing
* Add to cart
* Remove from cart
* Cart total calculation
* Responsive UI

---

Tech Stack

* Frontend: React,HTML,CSS, Typescript

---

 Installation & Setup
 1. Clone the repository

git clone https://github.com/YOUR_USERNAME/e-commerce-cart-app.git
cd e-commerce


### 2. Install dependencies

npm install


### 3. Run the application

npm start

4. Open in browser


http://localhost:3000
